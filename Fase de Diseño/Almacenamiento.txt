ALMACENAMIENTO:
	Se carga todo al iniciar la aplicación
	Habrá una serie de almacenes para cada cosa que haya que guardar:
		1. almacén personajes
		2. almacén usuarios
		3. almacén desafíos/combates
		4. almacén equipo
		5. almacén ranking
		6. almacén esbirros
		7. almacén modificadores
		8. almacén habilidades especiales


	Cuando se carga todo al inicio se guarda en una variable llamada "State" o similar,
	que contiene toda esta información, para que sea fácil acceder a ella y pasarsela a otras funciones
	
	La idea es que se guarden los cambios según ocurren. 
	IMPORTANTE: no se guarda todo State cada vez que ocurra un cambio, sólo el almacén modificado
	IMPORTANTE: se hay que asegurar de que se cambian los datos en los archivos y en State.
